Guia completo para cadastrar itens no sistema

Introdução

Este guia detalhado vai ajudar você a cadastrar seus itens corretamente no sistema Estoque 2D, garantindo que seu estoque físico e digital estejam sempre sincronizados e organizados.

Dica: Um cadastro bem feito economiza tempo na busca e evita perda de itens!

Campos do Formulário

Nome do Item

Insira um nome descritivo e claro. Exemplos: "Parafuso sextavado M5x20", "Chave de fenda Philips #2"

Referência

Código único do item. Pode ser do fabricante ou interno: "REF1234", "ABC-009", "PFS-M5-20"

Localização no Estoque

O sistema usa um padrão de codificação para identificar exatamente onde cada item está armazenado. A localização é composta por três partes:

C2P3A1
C2

Corredor 2

P3

Prateleira 3

A1

Altura 1

Importante: Mantenha a mesma codificação no estoque físico, colocando etiquetas visíveis em cada corredor, prateleira e altura.

Etiquetas para o Estoque Físico

Padronização é Fundamental

Para garantir que seu estoque físico corresponda exatamente ao digital, recomendamos:

  • Imprimir etiquetas com os códigos de localização
  • Colar nos locais correspondentes no estoque
  • Usar cores diferentes para cada tipo de área (corredor, prateleira, altura)
  • Manter as etiquetas sempre visíveis e legíveis
